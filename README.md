# Página Web Personal

Este es mi primer proyecto de una página web personal creada con HTML y CSS.

## Contenido de la página

- Foto personal
- Nombre y apellido
- Dirección
- Número de teléfono
- Correo electrónico
- Hobbies
- Redes sociales
- Mapa de ubicación
- Video de YouTube

## Tecnologías utilizadas

- HTML5
- CSS3
